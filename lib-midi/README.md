# Library MIDI
## Open Source C++ library for the MIDI implementation

Supported platforms:

- Board's: Orange Pi Zero / One (baremetal) {Allwinner SoC H2+/H3 ARM}
- MCU's: GD32F103R / GD32F107R / GD32F207R / GD32F407R {GigaDevice MCU ARM}
